Introduction and Background¶

Disability and Public Health by Jackson Scrimpshire and Nikhil Modayur. The GitHub repository can be found here.

Description of Project:

This project will seek to understand questions related to the DHDS database collected by the CDC, and assess possible correlations with other public health datasets including tobacco usage. Possible questions that will be investigated include:

  • What is the percentage of adults with disabilities in each state?

  • What is the percentage of adults with specific functional disability types in each state?

  • How does the percentage of adults with disabilities vary by age, sex, and race/ethnicity in each state?

  • How does the state percentage compare with the national percentage? With percentages in neighboring states?

  • Does the percentage of adults with key health indicators, such as obesity, physical activity and smoking vary across different functional disability types? Among adults with and without disabilities?

Datasources:

  1. DHDS - Prevalence of Disability Status and Types. Disability and Health Data System (DHDS) is an online source of state-level data on adults with disabilities. Users can access information on six functional disability types: cognitive (serious difficulty concentrating, remembering or making decisions), hearing (serious difficulty hearing or deaf), mobility (serious difficulty walking or climbing stairs), vision (serious difficulty seeing), self-care (difficulty dressing or bathing) and independent living (difficulty doing errands alone).
  2. BRFSS Prevalence and Trends Data: Tobacco Use - Four Level Smoking Data for 1995-2010

Collaboration Plan: We have a shared Google Colab and GitHub where we will push our code to each other. We will also coordinate via text messages and in person meetings when needed. At minimum, we will touch base regarding the project once a week.

Project Goals:

  1. Create a regression model that will predict the smoking values in future years

  2. Quantify relationships between disability status and health behaviors (smoking, physical activity, obesity) through hypothesis testing and regression models.

  3. Create interactive visualizations of disability prevalence across states

  4. Build predictive models to determine which demographic factors (age, sex, race/ethnicity) are most strongly associated with different disability types across states.

latest_year = dhds['Year'].max()

disability_types = [
    'Cognitive Disability',

    'Mobility Disability',
    'Vision Disability',

type_prevalence = []

    data = dhds[(dhds['Response'] == disability) & (dhds['Year'] == latest_year)]
    avg = data['Data_Value'].mean()
    type_prevalence.append({'Type': disability, 'Prevalence': avg})

# Convert to df anbd sort
type_df = pd.DataFrame(type_prevalence).sort_values('Prevalence', ascending=False)
